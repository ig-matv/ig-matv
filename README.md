##  Sobre Mim

Olá! Meu nome é **Igor Matvicio**, atuo como **Quality Assurance Engineer** com foco em validação de aplicações web e APIs.

Tenho experiência na elaboração de cenários de teste claros e estruturados utilizando **Gherkin (BDD)**, garantindo comunicação eficaz entre equipes técnicas e de negócio. Minha abordagem une qualidade, confiabilidade e alinhamento com os requisitos do projeto.

```gherkin

  Cenario: Garantia de qualidade em cada entrega
    Dado um requisito bem definido
    Quando Igor elabora e executa os cenários de teste
    Então a funcionalidade é validada com precisão
    E os bugs são reportados e rastreados no Jira
    E a qualidade do software é garantida ✅
```
