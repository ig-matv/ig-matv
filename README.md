## Sobre Mim

Olá! Meu nome é **Igor Matvicio**.

Atuo na validação de aplicações web e APIs, garantindo que cada funcionalidade seja testada com precisão, confiabilidade e alinhamento aos requisitos do projeto.

Possuo experiência na elaboração de cenários de teste claros e estruturados, utilizando **Gherkin (BDD)** para documentar comportamentos esperados de forma organizada e compreensível, facilitando a comunicação entre as equipes.
